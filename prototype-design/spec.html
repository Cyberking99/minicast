8. Icon & Visual Language
  • Icons use Feather-style thin 2px strokes — clean, not decorative
  • No emoji as feature icons (emoji allowed only for inline reactions like pool countdown indicators)
  • No rounded pill buttons on every action — only primary CTA uses full rounding
  • No glass morphism, no glowing neon on dark, no floating orbs
  • No gradient backgrounds on panels or cards — reserved for Oracle Verdict header only
  • Avatar = truncated address with last 2 hex chars as initials in a circle
